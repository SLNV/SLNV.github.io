题解 「土」秘法地震

链接: https://ac.nowcoder.com/acm/problem/53676?&headNav=acm

思路:

首先求出二维前缀和，然后遍历所有区间，区间值大于 0 就 ans++ ，最后输出 ans 即可。

求二维前缀和

可以一边读入一边算前缀和，用同一个数组存也可以，因为后面不需要初始值了。但要注意输入时数字之间没有空格，需要用 string 类型读入。

二维前缀和递推式： sum[i][j] = sum[i - 1][j] + sum[i][j - 1] - sum[i - 1][j - 1] + arr[i][j]

遍历数组

遍历算出区间和，大于 0 则代表区间中有 1 ，否则则全是 0。

区间值计算式： div[i][j] = sum[i][j] - sum[i - k][j] - sum[i][j - k] + sum[i - k][j - k]
